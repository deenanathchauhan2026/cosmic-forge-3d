# Cosmic Forge — Interactive Three.js Animation

A responsive Vite + Three.js single-page web app with orbit controls, animated rings, crystal shards, color-shifting lights, bloom, and particle effects.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

This project includes `vercel.json` and builds to `dist`.

```bash
vercel --prod
```

Or connect the GitHub repository to Vercel and use the default build command: `npm run build`.
